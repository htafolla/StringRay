# StrRay Lite Extension

StrRay Lite provides essential AI-assisted development capabilities within the oh-my-opencode ecosystem, optimized for teams that need quality safeguards without full framework complexity.

## Relationship to Full Framework

- **Base**: Built on oh-my-opencode v1.1.1 (same as full version)
- **Agents**: 4 core agents (vs 8 in full version)
- **Configuration**: Uses same oh-my-opencode.json schema
- **Models**: Same "opencode/grok-code" integration
- **Upgrade Path**: Can migrate to full framework when needed
